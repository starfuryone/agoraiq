Overview

The AgoraIQ REST API provides programmatic access to signal intelligence data, provider scores, market data, and proof records. API access is available on the ELITE plan.

Base URL: https://api.agoraiq.net/api/v1

All requests must use HTTPS. The API returns JSON. Authentication uses HTTP-only session cookies.

Authentication

Authenticate by POSTing credentials to /api/v1/auth/login. A session cookie is returned which must be included in all subsequent requests.

POST /api/v1/auth/login
Content-Type: application/json

{
  "email": "you@example.com",
  "password": "yourpassword"
}

Include credentials: 'include' in all fetch requests to send the session cookie.

Rate Limits

API requests are rate-limited to 100 requests/minute per user on ELITE tier. Rate limit headers are returned on every response:

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 87
X-RateLimit-Reset: 1706000060

Signals

GET /api/v1/signals/recent

Returns the most recent signals across all tracked providers.

GET /api/v1/signals/recent?limit=50&provider=PROVIDER_ID

Providers

GET /api/v1/providers

Returns all tracked signal providers with their scores and statistics.

GET /api/v1/providers?sort=score&verified=true

Cornix Signal Format

AgoraIQ natively parses signals in Cornix format. The standard format is:

#BTCUSDT LONG

Entry: 42000 - 42500
Targets: 43000 / 44000 / 46000 / 50000
Stop: 40500

Leverage: 10x

Fields parsed: pair, direction, entry zone, take-profit targets (TP1–TP4), stop loss, leverage. LLM fallback parsing is used for non-standard formats.

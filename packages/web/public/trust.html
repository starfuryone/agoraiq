Trust & Safety
How AgoraIQ protects users and maintains data integrity.

Proof System

🔒
SHA-256 Immutable Hashing
Every signal is hashed with SHA-256 the moment it is received by our system. The hash, timestamp, and original content are stored in an append-only database. No administrator, moderator, or provider can alter historical signal records. You can verify any signal using the hash verification tool on the Proof Dashboard.

Cherry-Pick Detection

🕵️
Algorithmic Manipulation Detection
Our detection algorithms analyse signal patterns for signs of selective reporting. Providers with suspiciously high win rates relative to signal volume, unusual outcome timing, or missing outcome reports are flagged and their scores are adjusted accordingly.

Platform Security

AgoraIQ implements industry-standard security practices:

  • All data transmitted over HTTPS with TLS 1.3
  • Passwords hashed with bcrypt (cost factor 12+)
  • CSRF protection on all authenticated endpoints
  • Rate limiting on auth and API endpoints
  • HTTP-only, SameSite=Strict session cookies
  • Regular dependency audits and security patches
